# get-reward

Take out all the rewards

# usage

```sh
get-reward <TOKEN_SYMBOL>
```

- `<TOKEN_SYMBOL>`: can be one of the following
  
    - `UNI-V2[USDT]`
    - `UNI-V2[WETH]`
    - `UNI-V2[WBTC]`
    

# examples

- get `UNI-V2[WBTC]` liquidity token rewards

    ```sh
    get-reward UNI-V2[WBTC] 
    ```
