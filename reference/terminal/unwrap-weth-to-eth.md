# unwrap-weth-to-eth

unwrap WETH to ETH

must connect wallet first

# usage

```sh
unwrap-weth-to-eth [amount] 
```

- `[amount]`: the amount to unwrap WETH to ETH; default value is 0

# examples

- unwrap `0` `WETH` to `ETH` (default)

    ```sh
    unwrap-weth-to-eth
    ```

- unwrap `100` `WETH` to `ETH`

  ```sh
  unwrap-weth-to-eth 100
  ```
